# PCB Layout

## Board Specification

- Width: 105mm
- Height: 80mm
- Corner radius: 2mm
- Mounting holes: 2mm diameter, one at each corner, 3.5mm from each edge

## Orientation

All directions described as viewed from the front (LCD-facing) side.

## Component Placement

- J3 (FPC connector): front side, left edge
- J1 (USB-C): back side, right edge, as close to ESP32 as possible
- J2 (Battery connector): back side, top-left edge
- U1 (ESP32-S3-WROOM-1U): antenna extends outside the frame
- SW1 (BOOT) + SW2 (RESET): back side, side by side, near ESP32, clickable from frame
